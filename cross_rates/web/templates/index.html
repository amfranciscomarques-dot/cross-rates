Enquadramento teórico

1. Convenção A/B

A = base (ao certo, 1 unidade); B = cotada (ao incerto, valor variável). O preço diz quantas unidades de B valem 1 A. Mostra-se bid – ask, bid ≤ ask.

2. Regra de ouro do bid/ask

• Banco/market maker: compra a base ao bid, vende ao ask.
• Cliente/empresa: compra a base ao ask, vende ao bid.
• Empresa: recebe ME → vai vender; paga ME → vai comprar.
Spread = ask − bid (a margem do banco).

3. Cross-rate (taxa cruzada)

Câmbio entre duas moedas sem cotação direta, via uma moeda-veículo C. Em cada conversão aplica-se a ponta desfavorável ao cliente — passar por uma moeda intermédia nunca contorna a margem.

Cross direto (÷) — C do mesmo lado: X/Y = (X/C) ÷ (Y/C) bid = bid(X/C) ÷ ask(Y/C) ; ask = ask(X/C) ÷ bid(Y/C) Cross indireto (×) — C em lados opostos: X/Y = (X/C) × (C/Y) bid = bid(X/C) × bid(C/Y) ; ask = ask(X/C) × ask(C/Y)

4. Arbitragem geográfica

Mesmo par cotado em praças diferentes. Há ganho certo se o ask mais baixo (onde se compra a base) for inferior ao bid mais alto: ask(A) < bid(B) ⇒ compra em A, vende em B. Ganho na moeda cotada.

5. Arbitragem triangular

Inconsistência entre três pares. Um ciclo A→B→C→A cujo produto das taxas seja > 1 ⇒ ganho certo = (fator − 1) × montante. Risco nulo: pernas simultâneas.

6. Forward (PTJ)

Câmbio fixado hoje para uma data futura. Taxa de equilíbrio que impede a arbitragem com a réplica no mercado monetário:

F = S × (1 + i_cotada·n/360) ÷ (1 + i_base·n/360)

• i_cotada > i_base → F > S: base a prémio.
• i_cotada < i_base → F < S: base a desconto.
Arbitragem a prazo: se o forward de mercado sai do intervalo de equilíbrio, vende-se o caro e replica-se via MMI o barato.

7. Como esta ferramenta calcula

Cada cotação BASE/COTADA gera duas conversões dirigidas: BASE→COTADA à taxa bid; COTADA→BASE à taxa 1/ask. O bid do cross é o percurso BASE→…→COTADA; o ask é 1/(taxa COTADA→…→BASE).

8. Opções cambiais (Garman-Kohlhagen)

Opção vanilla sobre BASE/COTADA: a BASE é o ativo (rende r_f), a COTADA o numerário (rende r_d). Black-Scholes adaptado a divisas:

call = S·e^(−r_f·T)·N(d1) − K·e^(−r_d·T)·N(d2) d1 = [ln(S/K) + (r_d − r_f + σ²/2)·T] / (σ·√T) ; d2 = d1 − σ·√T

O prémio vem em COTADA por 1 BASE. As Gregas medem a sensibilidade: delta (spot), gamma (delta), vega (vol), theta (tempo), rho (taxa doméstica). Vale a paridade put-call C − P = S·e^(−r_f·T) − K·e^(−r_d·T).
